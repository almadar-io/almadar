# مهارات المدار

عزز سير عمل التطوير لديك بمهارات مدعومة بالذكاء الاصطناعي لتوليد مخططات المدار.

## almadar-orbitals

المهارة الأساسية لتوليد مخططات المدار من وصف باللغة الطبيعية.

### ماذا تفعل؟

- توليد ملفات مخطط المدار (.orb) كاملة من الأوصاف
- إنشاء كيانات بأنواع الحقول والعلاقات الصحيحة
- تصميم آلات الحالة مع الحراس والتأثيرات
- إنتاج أنماط واجهة المستخدم وتخطيطات الصفحات
- اتباع أفضل ممارسات المدار

### التثبيت

```bash
# عبر أداة المدار
almadar skill install almadar-orbitals

# أو التحميل اليدوي
curl -O https://almadar.io/skills/almadar-orbitals.zip
unzip almadar-orbitals.zip -d ~/.almadar/skills/
```

### الاستخدام

بعد التثبيت، استدعِ المهارة في جلستك:

```
/almadar-orbitals ابنِ نظام إدارة مهام مع مصادقة المستخدمين،
                  تعيين المهام، وسير عمل الموافقات
```

المهارة ستولد:

```json
{
  "name": "نظام_إدارة_المهام",
  "orbitals": [
    {
      "name": "المستخدمون",
      "entity": { ... },
      "traits": [{ "ref": "مصادقة_المستخدم" }, { "ref": "ملف_المستخدم" }]
    },
    {
      "name": "المهام",
      "entity": { ... },
      "traits": [{ "ref": "دورة_حياة_المهمة" }, { "ref": "تعيين_المهمة" }]
    },
    {
      "name": "الموافقات",
      "entity": { ... },
      "traits": [{ "ref": "سير_عمل_الموافقة" }]
    }
  ]
}
```

### المحتويات المضمنة

| الملف | الوصف |
|-------|-------|
| `SKILL.md` | تعليمات المهارة وقدراتها |
| `examples/` | مخططات مرجعية للأنماط الشائعة |
| `validation/` | رموز الأخطاء وقواعد التحقق |
| `patterns/` | مرجع أنماط واجهة المستخدم |
| `traits/` | مرجع مكتبة السمات القياسية |

### أفضل الممارسات

1. **كن محدداً** - تفاصيل أكثر = مخططات أفضل
2. **صِف سير العمل** - "المستخدمون يمكنهم تقديم ← موافقة ← نشر"
3. **اذكر الأدوار** - "المديرون يمكنهم الحذف، المستخدمون يمكنهم العرض فقط"
4. **حدد التخزين** - "المهام مخزنة، حالة اللعبة في الذاكرة"

### أمثلة على الطلبات

**التجارة الإلكترونية:**
```
ابنِ منصة تجارة إلكترونية مع:
- كتالوج منتجات مع تصنيفات
- سلة تسوق (حالة مؤقتة)
- سير عمل الدفع مع تكامل الدفع
- إدارة الطلبات مع تتبع الحالة
- لوحة تحكم المدير للمخزون
```

**لعبة:**
```
ابنِ لعبة منصات ثنائية الأبعاد مع:
- لاعب بالصحة والأرواح والنقاط
- أعداء بذكاء اصطناعي للدوريات
- مقتنيات وقوى خاصة
- تقدم المراحل
- لوحة المتصدرين
```
