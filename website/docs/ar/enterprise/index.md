# المدار للأعمال

> **رؤية بشرية، تنفيذ ذكي** — نبني تطبيقاتك بنهج المدار

## ما نقدمه

المدار هي مبتكرة لغة المدار ووكالة تطوير برمجيات متكاملة الخدمات. نستخدم تقنيتنا الخاصة لتسليم المشاريع بشكل أسرع وأفضل وأكثر فعالية من التطوير التقليدي.

### للشركات والمؤسسات

| الخدمة | الوصف |
|--------|-------|
| **التطوير المخصص** | تطبيقات متكاملة مبنية بنهج المدار |
| **خدمات الترحيل** | تحويل الأنظمة القديمة إلى بنية المدار الحديثة |
| **التدريب وورش العمل** | تدريب في الموقع أو عن بُعد لفريقك |
| **الاستشارات** | مراجعة البنية التقنية، الاستشارات الفنية |
| **الدعم ذو الأولوية** | دعم باتفاقية مستوى خدمة مع مهندسين مخصصين |

### لماذا تختار المدار؟

| الوكالة التقليدية | المدار |
|------------------|--------|
| فرق كبيرة (٥-١٠ أشخاص) | فريق رشيق (خبيران) |
| شهور من التطوير | أسابيع للإنتاج |
| طلبات تغيير مكلفة | تغييرات مرنة وفعالة التكلفة |
| التوثيق كفكرة لاحقة | التوثيق مدمج في العملية |
| برمجة يدوية ١٠٠٪ | تطوير متسارع بالذكاء الاصطناعي |

## نهجنا

```
متطلباتك
    ↓
الاكتشاف والتصميم (١-٢ أسبوع)
    ↓
تطوير المدار (٢-٦ أسابيع)
    ↓
النشر للإنتاج
    ↓
الدعم المستمر
```

### ما تحصل عليه

- **الكود المصدري الكامل** — React/TypeScript قياسي، بدون قفل مع المورد
- **توثيق كامل** — البنية، API، أدلة المستخدم
- **نقل المعرفة** — فريقك يمكنه الصيانة والتطوير
- **دعم ٦٠ يوماً** — دعم ما بعد الإطلاق مشمول

## دراسات الحالة

### التحول الرقمي الحكومي

**العميل:** وزارة التفتيش (سلوفينيا)  
**التحدي:** رقمنة ٥٠+ سير عمل تفتيش عبر إدارات متعددة  
**الحل:** نظام إدارة التفتيش المبني على المدار  
**النتائج:**
- ١٢ وحدة تم تسليمها في ٨ أسابيع
- تخفيض التكلفة ٦٠٪ مقارنة بالتقدير التقليدي
- امتثال كامل للوائح الاتحاد الأوروبي

### منصة التدريب الرياضي

**العميل:** مدرب شخصي نخبة (السعودية)  
**التحدي:** إدارة عملاء مخصصة وتتبع التمارين  
**الحل:** تطبيق يركز على الجوال مع مزامنة فورية  
**النتائج:**
- MVP في ٣ أسابيع
- بنية قابلة للتوسع للنمو المستقبلي
- واجهة عربية أولاً

## القطاعات التي نخدمها

- **الحكومة والقطاع العام** — الامتثال، سجلات التدقيق، خدمات المواطنين
- **الرعاية الصحية** — إدارة المرضى، الجدولة، السجلات
- **المالية** — الخدمات المصرفية الإسلامية، التقنية المالية، منصات التداول
- **التجارة الإلكترونية** — المتاجر، المخزون، التوصيل
- **التعليم** — أنظمة إدارة التعلم، إدارة الطلاب، التقييمات
- **اللوجستيات** — إدارة الأسطول، التتبع، التحسين

## التركيز على السعودية ودول الخليج

نتخصص في المشاريع للسوق السعودي والخليجي:

- **التوافق مع رؤية ٢٠٣٠** — خبرة في التحول الرقمي
- **العربية أولاً** — دعم RTL أصلي، واجهات عربية
- **الامتثال المحلي** — ساما، الهيئة الوطنية للأمن السيبراني، واللوائح الإقليمية
- **دعم محلي** — فريق في المنطقة

## الأسئلة الشائعة

### هل أحتاج لتعلم المدار للعمل معكم؟

لا. نحن نتولى كل التنفيذ التقني. تحصل على كود قياسي وقابل للقراءة يمكن لأي مطور React/TypeScript صيانته.

### ماذا لو أردت متابعة التطوير داخلياً؟

نقدم نقل معرفة كامل. فريقك يحصل على التوثيق والتدريب والقدرة على توسيع النظام بشكل مستقل.

### هل تقدمون صيانة مستمرة؟

نعم. نقدم خطط اشتراك للتطوير المستمر وإصلاح الأخطاء وإضافة الميزات.

### هل يمكنكم العمل مع أنظمتنا الحالية؟

بالتأكيد. نتخصص في التكامل والترحيل. يمكننا ربط تطبيقات المدار بواجهات API وقواعد البيانات والخدمات الحالية.

### ماذا عن الملكية الفكرية؟

أنت تملك كل شيء. الكود المصدري والتوثيق وجميع المخرجات ملكك بدون قيود.
