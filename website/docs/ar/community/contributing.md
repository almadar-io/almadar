# المساهمة في المدار

أهلاً بك في مجتمع المدار! نحن متحمسون لمساهمتك.

## طرق المساهمة

### التوثيق

- إصلاح الأخطاء الإملائية وتحسين الوضوح
- ترجمة المحتوى إلى العربية أو لغات أخرى
- كتابة الدروس والأدلة
- إضافة أمثلة وحالات استخدام

### المكتبة القياسية

- اقتراح عوامل جديدة
- تحسين التطبيقات الحالية
- إضافة الاختبارات والمقاييس

### الأنماط والسمات

- تقديم أنماط واجهة مستخدم جديدة
- إنشاء سمات قابلة لإعادة الاستخدام
- توثيق أفضل الممارسات

### المجتمع

- الإجابة على الأسئلة في Discord
- كتابة مقالات المدونة
- إنشاء دروس فيديو
- إلقاء محاضرات في اللقاءات

## عملية المساهمة

### ١. للتوثيق

```bash
# انسخ المستودع واستنسخه
git clone https://github.com/YOUR_USERNAME/almadar-docs.git

# أنشئ فرعاً
git checkout -b docs/improve-getting-started

# أجرِ تغييراتك
# ...

# أرسل طلب دمج
```

### ٢. للكود (المكتبة القياسية، الأنماط)

1. **افتح RFC** - للتغييرات الكبيرة، ابدأ نقاشاً
2. **احصل على ملاحظات** - انتظر مدخلات المجتمع
3. **نفّذ** - بعد الموافقة، أرسل PR
4. **مراجعة** - عالج ملاحظات المشرفين
5. **دمج** - احتفل!

### ٣. للترجمات

نستخدم سير عمل للترجمة:

```bash
# الترجمات العربية تذهب إلى مجلد /ar/
docs/
├── en/
│   └── getting-started/
│       └── introduction.md      # المصدر
└── ar/
    └── getting-started/
        └── introduction.md      # الترجمة
```

**إرشادات الترجمة:**
- حافظ على اتساق المصطلحات التقنية (انظر القاموس أدناه)
- احتفظ بكتل الكود باللغة الأصلية
- استخدم تنسيق RTL المناسب
- اختبر العرض قبل الإرسال

## قواعد السلوك

### تعهدنا

نلتزم بتوفير بيئة ترحيبية وشاملة للجميع، بغض النظر عن:

- الخلفية أو مستوى الخبرة
- الهوية أو التعبير الجنسي
- الموقع الجغرافي
- اللغة
- الدين

### السلوك المتوقع

- كن محترماً وشاملاً
- رحب بالقادمين الجدد
- قدم ملاحظات بناءة
- ركز على المشكلة، لا الشخص

### السلوك غير المقبول

- التحرش أو التمييز
- التصيد أو التعليقات المهينة
- نشر معلومات خاصة للآخرين
- أي سلوك آخر يُعتبر غير لائق

### التنفيذ

المخالفات قد تؤدي إلى:
1. تحذير
2. حظر مؤقت
3. حظر دائم

للإبلاغ عن المشاكل: conduct@almadar.io
