يسعدنا تقديم **المدار** - فيزياء البرمجيات.

المدار هو إطار عمل تصريحي لبناء تطبيقات متكاملة من خلال آلات الحالة. عرّف كياناتك وسلوكياتك وواجهتك كمخططات تُترجم إلى كود جاهز للإنتاج.

<!-- truncate -->

## ما هو المدار؟

المدار (بالعربية: المدار) يجلب نموذجاً جديداً لتطوير البرمجيات:

- **مخططات تصريحية**: عرّف بنية تطبيقك بالكامل في مكان واحد
- **آلات الحالة**: صمم السلوك كآلات حالة قابلة للتنبؤ والاختبار
- **توليد متكامل**: ترجم إلى واجهة React وخادم Express/FastAPI ونماذج قاعدة البيانات

## نموذج المدار

كما تدور الإلكترونات حول النواة وفق قواعد الكم، تتبع مكونات تطبيقك قواعد آلة الحالة. كل **مدار** هو كيان مع **سمات** مرفقة تحدد سلوكه وواجهته وتكاملاته.

```json
{
  "orbitals": [{
    "name": "TaskManager",
    "entity": {
      "name": "Task",
      "fields": [
        { "name": "title", "type": "string" },
        { "name": "status", "type": "enum", "options": ["todo", "done"] }
      ]
    },
    "traits": [
      { "ref": "Listable" },
      { "ref": "Editable" }
    ]
  }]
}
```
