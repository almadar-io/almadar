# مقدمة إلى المدار

> **فيزياء البرمجيات**: لغة طبيعية ← مخطط ← تطبيق إنتاج

## ما هو المدار؟

المدار (Almadar) هو **لغة برمجة تصريحية** لبناء تطبيقات متكاملة. بدلاً من كتابة كود إلزامي متناثر عبر الواجهة الأمامية والخلفية، تصرّح بتطبيقك كمخطط من:

- **الكيانات** - هياكل البيانات مع قواعد الاستمرار
- **السمات** - السلوك المعرّف كآلات حالة
- **الصفحات** - المسارات مع ربط الواجهة

## لماذا "المدار"؟

المدار يعني "Orbit" بالإنجليزية. مثلما تتبع الكواكب مسارات متوقعة تحكمها قوانين الفيزياء، تتبع التطبيقات المبنية بالمدار مسارات متوقعة تحكمها آلات الحالة.
