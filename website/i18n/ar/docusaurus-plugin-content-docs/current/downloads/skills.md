# مهارات المدار

مهارات مدعومة بالذكاء الاصطناعي لتوليد مخططات المدار من أوصاف اللغة الطبيعية.

## المهارات المتاحة

### almadar-orbitals

توليد مخططات المدار الكاملة من المتطلبات.

### almadar-fixing

إصلاح أخطاء التحقق في المخططات الموجودة.

## الاستخدام

صف متطلبات تطبيقك بلغة طبيعية، وستقوم المهارة بتوليد مخطط المدار صالح.
