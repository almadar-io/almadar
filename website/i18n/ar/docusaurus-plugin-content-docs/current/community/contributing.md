# المساهمة في المدار

نرحب بالمساهمات من المجتمع!

## طرق المساهمة

- **الإبلاغ عن الأخطاء** - افتح issues على GitHub
- **اقتراح الميزات** - ابدأ نقاشاً
- **تقديم PRs** - إصلاح الأخطاء أو إضافة ميزات
- **كتابة التوثيق** - تحسين الوثائق
- **المشاركة** - أخبر الآخرين عن المدار

## البدء

1. Fork المستودع
2. أنشئ فرعاً للميزة
3. قم بتغييراتك
4. قدم pull request
