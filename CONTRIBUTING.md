# Contributing to Almadar

Thank you for your interest in contributing to Almadar!

## Ways to Contribute

### Documentation
- Fix typos and improve clarity
- Translate content to Arabic or other languages
- Write tutorials and guides
- Add examples

### Standard Library
- Propose new operators
- Improve existing implementations
- Add tests

### Patterns
- Submit new UI patterns
- Document best practices

### Community
- Answer questions on Discord
- Write blog posts
- Create video tutorials

## Process

1. **Fork** the repository
2. **Create a branch** for your changes
3. **Make your changes**
4. **Test** your changes
5. **Submit a pull request**

## Code of Conduct

Please read our [Code of Conduct](./CODE_OF_CONDUCT.md) before contributing.

## Questions?

- Join our [Discord](https://discord.gg/almadar)
- Open a [GitHub Discussion](https://github.com/almadar-io/almadar/discussions)
- Email: community@almadar.io
